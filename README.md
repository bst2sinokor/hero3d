# 107 — 그린파워 부지 vs 웅동 항만배후부지 비교 애니메이션

## 프로젝트 개요

부산그린파워 산업폐기물 매립 부지와 웅동2단계 항만배후부지의 **지반 안정화 방식 차이**를 시네마틱 3D로 대조하는 웹 인터랙티브 애니메이션.

- **핵심 메시지**: 그린파워 부지는 차수 라이너로 폐기물을 가둔 구조라, 웅동처럼 단순히 성토·PBD 압밀만으로는 라이너 아래 원지반이 침하하며 라이너를 손상시킬 수 있음. 두 부지의 지반 처리 방식 차이(웅동=선행재하·PBD 압밀 / 그린파워=차수성 위생매립)를 씬별로 시각화하여, 그린파워에는 **라이너를 지키는 맞춤 공법**이 필요함을 강조.
- **대상 청중**: 80세 회장님 — 큰 자막, 한 장면 한 메시지, 쉬운 비유, 긍정적 톤, 자막만(음성 없음).
- **산출물 형태**: 웹 인터랙티브(GSAP 타임라인 자동재생) + 필요 시 화면 녹화 영상.
- **실행 환경**: 브라우저(로컬 파일 또는 정적 호스팅). 빌드 도구 불필요.

## 핵심 컴포넌트

### 1. 지층·좌표계 (양쪽 공통)
- **World scale**: Wd=2400(가로), Dp=1400(깊이), GROUND=40(지표), SOFT_BOT=-420(연약점토층 하부=기반암 상부)
- **지층**: 매립층(준설토, 상단) → 연약점토층(중간) → 기반암(하단) — 각 층은 ExtrudeGeometry + PBR 텍스처
- **지층 라벨**: HTML overlay(`layerTag`)로 3D 앵커 좌표를 화면 좌측에 지시선 연결

### 2. 그린파워 셀·라이너 (씬 6~8)
- **PIT**: 사다리꼴 셀 (상단 반폭 900, 하단 반폭 520, 깊이 290) — 위→아래 클리핑으로 굴착 표현
- **4중 라이너**: 자갈층·상부 부직포·차수 지오멤브레인·하부 부직포 순차 적층
- **PBD·멤브레인**: 씬 7 진공압밀 (hero3d.html과 동일 원리, 차수 셀 내부에서 수행)

### 3. 씬 13 침하·라이너 파단·오염수 누출 (최근 구현)
- **hero3d.html의 성토 침하 패턴 재사용**: `settleBump` + `applyWHSettle`로 매립층·연약점토층·라이너·pool 정점을 국부 변위(v: 0→50)
- **크랙**: 6개 (WH_X±215~130), 라이너 앞면에 사다리꼴 파단면 → 갈색 흙 fill
- **오염수 유출 (실시간 흐름 애니메이션)**: 5개 mesh로 분리
  - Mesh 0: 크랙 채움 (opacity fade-in)
  - Mesh 1: 크랙 → 연약점토층 다각도 침투 (4경로/크랙: 수직·좌 lateral·우 lateral·깊은 V-spread), clip plane으로 위→아래 reveal
  - Mesh 2: 기반암 상부 pool 밴드
  - Mesh 3: 좌측 확산 (clip plane center→left reveal, x=185→-960)
  - Mesh 4: 우측 확산 (clip plane center→right reveal, x=530→1090)
- **동기 pool 수위 하강**: `g_poolClip.constant` 7초 동안 POOL_PARTIAL → POOL_BOT-6
- **재질 통일**: pool과 동일한 MeshStandardMaterial 파라미터 (color 0xaadc3a, emissive 0x4a6a12, roughness 0.25, opacity 0.82)

### 4. 자막·씬 라벨
- 각 씬 상단 자막(`#hud .cap`)이 GSAP `.call()`로 시간축에 배치, `<span class="hl">`로 키워드 강조.
- 좌측 하단 `#scenelabel`은 현재 씬 이름 표시.
